版本要求

vim 版本>=7.3 低版本可以执行 Vim74Install.sh 安装

查看vim版本  vim --version

安装 步骤

git clone https://github.com/wming3/.vimToIDE.git  ~/.vim

cd ~/.vim

sh Install.sh

新增插件

cd ~/.vim

git submodule add https://github.com/scrooloose/nerdcommenter.git bundle/nerdcommenter
